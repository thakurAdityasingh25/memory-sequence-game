# Memory Sequence Game (Simon Says)

## About the Project
This is a simple browser-based memory game inspired by the classic Simon Says game.  
The game tests the user’s memory by generating a random sequence that increases in difficulty with each level.

---

## Tech Stack
- HTML
- CSS
- JavaScript

---

## Features
- Random sequence generation
- Increasing difficulty levels
- User input validation
- Game over and restart functionality
- Interactive UI with button animations

---

## How to Play
1. Press any key to start the game  
2. Watch the sequence of colors  
3. Repeat the sequence in the same order  
4. The sequence gets longer with each level  
5. Game ends if you make a mistake
